# World Data

This directory contains world data:

- **world/**: Default world folder
- **world_nether/**: Nether dimension (if enabled)
- **world_the_end/**: End dimension (if enabled)
- Custom world folders

Each world folder contains:
- **region/**: Region files (.mca)
- **data/**: World data files
- **playerdata/**: Player data files
- **level.dat**: World level data
