# Server Runtime Directory

This directory contains runtime files for the AyuMC server.

## Directory Structure

- **config/**: Server configuration files (server.properties, etc.)
- **plugins/**: Compiled and installed plugin files
- **worlds/**: World data and save files
- **logs/**: Server log files
- **cache/**: Temporary cache files
- **crash-reports/**: Crash report files

## Note

This directory is for runtime files only. It is not a Dart package and does not contain a `pubspec.yaml` file.
